# Platform Account Merchandise Ecosystem

A full-stack account merchandise web application featuring a Flask server infrastructure, dynamic Jinja2 frontend template routing, and an integrated MySQL transaction engine.

---

## 📁 System Architecture
```text
platform_account_merchandise/
├── app.py                 # Core Flask application entry point & API controllers
├── templates/             # Jinja2 HTML templates directory
│   ├── index.html         # Main dashboard interface
│   ├── admin.html         # Management control panel
│   ├── login.html         # Secure access credential form
│   ├── checkout.html      # Secure checkout portal
│   └── receipt.html       # Transaction statement output
└── README.md              # Documentation and Deployment guide
